Browser Transformers PoC

Vite + vanilla JavaScript + transformers.js で、ブラウザ内AI推論を試す最小PoCです。

注意事項

  • 初回実行時はモデルをダウンロードするため時間がかかります。
  • モデルサイズや回線状況、端末性能で処理時間は変わります。
  • 推論はブラウザ内で実行され、入力テキストは外部LLM APIへ送信しません。
  • ただしモデルファイル取得のため Hugging Face Hub への通信は発生します。
